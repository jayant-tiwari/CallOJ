Create A Contest



Contest Code:

  • Contest code is a unique identifier for each contest. Keep a one-word contest code for the contest.
    If duplicacy is detected, you will be prompted to use a new code for the same.


    • Contest Name:

  • Contest name specifies the contest. Use a fancy name to stand out


    • Start Date:

  • Select from the drop down menu to use the required date to start the contest.


    • Duration:

  • Select from the drop down menu to use the required duration for the contest


    • Marking Style:

  • Select the marking style for scoring the contest. Refer to creating a problem documentation for more
    information. Even if you select a problem with marking belonging to another category, the marking style
    of contest overpowers the pattern of a single question


    • Add Problems to the contest:

  • Select the problems from the list below. Once the contest is created, there is no provision currently of
    editing problem or contest once created. For more, refer to the readme
